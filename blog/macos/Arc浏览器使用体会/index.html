总结

把总结放在前面： 我的使用 arc 浏览器的感觉就是两个字：舒服！（个人观点仅供参考） 优点：

  • 外观花哨（有好有坏吧）
  • 画架功能
  • 快捷键可以实现更简单的分屏，新页面等等
  • 调试方便
  • 多用户切换方便 缺点：
  • 性能开销大（笔记本可能会影响续航，如果外出还是 safari）
  • 有一些 bug（比如总是提示翻译网页）
  • 没有书签功能（对应的 pin 功能，不过很难用）

新奇的设计和花哨的外观，真的可以先体验一下，看看适不适合你。 当然如果习惯不同也不要勉强，适合自己的才是对的。

心路历程

其实我很早就有关注到这个浏览器，也早早参与了测试（好像是 2021 年吧，有点模糊），不过当时第一次打开然后 CPU 飙升，风扇狂扇，本来清爽的 macbook 直接煎蛋，给我留下了不小的心理阴影，然后惊恐的删掉了。 这次偶然间在 b 站看到有人推荐，看着让人舒适的界面，我最后还是下载了使用。之前的发热问题也没了（可能也和系统版本有关），可以使用了。

常用快捷键

按键功能
command+s打开左侧 siderbar
ctrl+⬆️+=新建分屏
ctrl+数字切换空间
command+数字切换空间中的页面
command+⬆️+L打开资料库
command+⬆️+C复制当前 url
command+W关闭当前页面(设置里叫归档)
command+F搜索
command+L打开 command 输入

画架

我觉得 arc 最优秀的功能就是这个画架！如果你有用 arc 一定要尝试！！
