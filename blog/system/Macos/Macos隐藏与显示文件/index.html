macos命令使用 chflags

最近在使用obsidian的时候，出现了文件会变成隐藏，目前原因不明，最后使用chflags nohidden命令将它们取消隐藏。

使用man chflags可以了解大致使用方式

#常用指令
chflags hidden  # 隐藏单一文件或文件夹
chflags nohidden # 取消隐藏单一文件或文件夹
chflags -R hidden # 隐藏或取消隐藏目录所有文件
