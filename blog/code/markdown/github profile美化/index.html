好像是去年更新的功能，今天整理 github 文件顺便尝试一下。

原理就是新建一个和你的 github 账号名一样的仓库，然后这个仓库中的 read me 就会显示在 profile 页面。

这是一个 md 文件，可以使用 markdown，也可以使用 html+css 来制作样式。

也有一些相关美化的项目可以用来搭配使用

  • skillicons 添加小图标
  • github-readme-stats 你的 github 状态
