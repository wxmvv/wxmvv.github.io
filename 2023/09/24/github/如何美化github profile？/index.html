如何美化github profile？

 2023/09/24 github2

我要变美！

好像是去年更新的功能，今天整理github文件顺便尝试一下。

原理就是新建一个和你的github账号名一样的仓库，然后这个仓库中的read me就会显示在profile页面。

这是一个md文件，可以使用markdown，也可以使用html+css来制作样式。

也有一些相关美化的项目可以用来搭配使用

  • skillicons 添加小图标
  • github-readme-stats 你的github状态
