hexo部署静态博客

 2020/11/03 hexo3, markdown3

前言

最近购买下载了IA writer，确实是一个十分干净的md编辑器，能激发自己写字的欲望。于是考虑重新开始博客部署。翻出了以前的hexo博客（大概18年部署的一直没更新，并且原来的设置文件都丢失了），整理一下重新部署。

安装hexo和cactus主题

hexo
cactus

hexo安装

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server   #启动服务

cactus主题安装

cd blog
git clone https://github.com/probberechts/hexo-theme-cactus.git themes/cactus

配置文件参考

hexo文档
cactus主题

使用方法

hexo new 文件名
hexo clean 或 hexo c 清除缓存
hexo generate 或 hexo g 生成静态文件
hexo server 或 hexo s 本地查看
hexo deploy 或 hexo d 上传
